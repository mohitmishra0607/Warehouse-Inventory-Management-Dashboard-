# Warehouse Inventory Management Dashboard
An Excel dashboard for warehouse inventory analysis.
## Features
- Total Products
- Total Inventory Value
- Reorder Required
- Low Stock Items
- Average Daily Demand
- Average Fulfillment Rate
- Inventory Turnover
- Total Orders

## Charts
- Stock by Category
- Inventory Value by Category
- Reorder Status
- Stock Status
- Orders by Warehouse Zone
- Forecast Demand vs Stock
  
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
- Slicers
